O que é uma Div?

Uma div é uma divisão na tela, como se fosse uma "caixa" a qual novos elementos serão inseridos dentro dela.

Neste exemplo, foram criadas três divisões na tela, cada uma contendo um texto. É possível, no entanto, criar infinitos elementos dentro de uma div, incluindo div dentro de outras div.

Note que, para cada div, é formado um retângulo na tela, representando o espaço ocupado por esta divisão na tela.

Note, também, que é possível colocar mais de um parágrafo dentro de uma mesma Div,

  • Assim
  • Como
  • também
  • é
  • possível
  • adicionar
  • uma
  • lista
